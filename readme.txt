Add translation for post meta data when using DeepL for WordPress. Integrates nicely with WooCommerce, Yoast SEO and major plugins.

== Installation ==

1. Go to 'Plugins > Add New' on your WordPress admin dashboard.
2. Search for 'Addon for Post Meta Translation using DeepL' plugin from the official WordPress plugin repository.
3. Click 'Install Now' and then 'Activate'.
4. That's it! You're all set to start translating your post meta.

== Description ==

As you may have already noticed, the <strong>DeepL for WordPress: translation plugin</strong> only translates the post <strong>title</strong>, <strong>content</strong> and <strong>excerpt</strong>. This means that vital post meta data is left out of most translations for users.

This plugin extends the functionality of the <strong>DeepL for WordPress: translation plugin</strong> by enabling you to translate post meta data easily during translations. It integrates nicely with the <strong>DeepL for WordPress</strong> & <strong>DeepL for WordPress pro</strong> plugin and is useful for translating post meta for popular plugins like <strong>WooCommerce</strong>, <strong>Yoast SEO</strong>, <strong>Easy Accordion</strong> and so on.

= ‚ú® Getting Started =

1. Make sure you already have <strong>DeepL for WordPress: translation plugin</strong> installed and activated.
2. Create a post or article and ensure it has post meta (you can use custom fields to do this).
3. Proceed to translate your post or article using <strong>DeepL for WordPress: translation plugin</strong>.
4. Observe that your post meta is also translated after the translation is completed by DeepL.

You can get a taste of how this plugin works, by using the [demo](https://tastewp.com/create/NMS/8.0/6.7.0/addon-for-post-meta-translation-using-deepl/twentytwentythree?ni=true&origin=wp) link.
