Translate post meta data when using DeepL translate.

== Installation ==

1. Go to 'Plugins > Add New' on your WordPress admin dashboard.
2. Search for 'Addon for Post Meta Translation using DeepL' plugin from the official WordPress plugin repository.
3. Click 'Install Now' and then 'Activate'.

== Description ==

Translate post meta data when using DeepL translate.
